# Image Editor

간단한 이미지 편집 프로그램입니다.

## 주요 기능

- 이미지 열기/저장 (PNG, JPEG, BMP, WebP, ICO 지원)
- 이미지 회전 (좌/우 90도)
- 좌우 반전
- 이미지 자르기
- 크기 조정
- 실제 크기로 보기
- 작업 취소 (최대 10단계)
- 마우스 위치의 픽셀 색상 정보 표시

## 단축키

- Ctrl+O: 열기
- Ctrl+S: 저장
- F5: 다시 열기
- Ctrl+Z: 실행 취소
- Ctrl+H: 좌우 반전
- Ctrl+L: 왼쪽으로 90도 회전
- Ctrl+R: 오른쪽으로 90도 회전
- Ctrl+X: 자르기
- Ctrl+E: 크기 조정
- F1: 프로그램 정보

## 시스템 요구사항

- Windows OS
- .NET 8.0 이상
